Tipos de imágenes para diseño gráfico

1. JPEG / JPG

Muy usado para fotos. Bueno para web. No tiene transparencias. Pierde calidad si se guarda muchas veces.

2. TIFF

Máxima calidad, sin pérdida. Archivos muy grandes. Usado en impresión.

3. PNG

Permite transparencias. Buena calidad. Ideal para logos, gráficos y web.

4. GIF

Permite animaciones. Solo 256 colores. Se ve mal en fotos detalladas.

5. Vectorial (SVG, AI, etc.)

No usa píxeles. Se escala sin perder calidad. Ideal para logos e ilustraciones. No sirve para fotos reales.

6. RAW

Formato de cámaras profesionales. Mucha calidad y editable. Archivos muy grandes.

7. BMP

Formato antiguo. Archivos muy pesados. Poco usado hoy en día.

8. PSD

Propio de Photoshop. Guarda capas y efectos. No es muy compatible fuera de Photoshop.

9. PDF

Formato universal. Combina imágenes, vectores y texto. No siempre sirve para editar imágenes.

10. SVG

Vectorial escalable. Ideal para web e iconos. No sirve para fotos con muchos detalles.

11. EPS

Vectorial profesional. Muy usado en impresión. Difícil de editar sin programas especiales.

12. WEBP

Moderno de Google. Archivos pequeños con buena calidad. Soporta transparencias y animaciones.

13. AVIF

Muy moderno. Mejor compresión que WebP. Se usa cada vez más en web.
